
# Yelp Dataset Instructions

Download the Yelp Open Dataset from:  
🔗 https://www.yelp.com/dataset  

After downloading, place the following files inside the `data/raw/` folder:
- `business.json`
- `review.json`
- `user.json`

Example folder structure:
data/
raw/
business.json
review.json
user.json
processed/
> ⚠️ Note: The `/data` folder is ignored by Git (.gitignore) to keep your repository lightweight and avoid licensing issues.
